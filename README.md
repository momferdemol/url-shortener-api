# url-shortener-api

Run program
```sh
$ uvicorn app.main:app --reload
```

**Dependency**

Python modules used in `pyproject.toml`.

**Environment**

Project uses `.env` file to load environment variables which overwrites `/app/config.py`.

**Development**

Local development environment created with `devenv.nix`.
